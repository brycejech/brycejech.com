basefront

basefront is a css micro-framework designed to facilitate rapid development while keeping a project as lean as possible.

basefront is small like Skeleton, and provides the same grid syntax as Bootstrap. This way, projects can stay lean and provide for a less painful replacement of basefront with Bootstrap if the need arises. The primary difference between basefront and Skeleton is that basefront does not use margin-based guttering.

This site was built using basefront as a jumping off point.
