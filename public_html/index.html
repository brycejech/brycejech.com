Hi, I'm Bryce

I am a full-stack web developer at the Oklahoma State University Foundation

I enjoy building data driven web applications and user interfaces. I specialize in JavaScript in the browser and NodeJS on the server. I gain particular enjoyment in finding creative ways of approaching business process automation and third party software integrations / plugins

I build custom solutions using JavaScript / NodeJS, Python, C#, PostgreSQL, MongoDB, and any other technologies that are a good fit for any given problem. I am particularly interested in RESTful web services, web security, and application architecture and design patterns

The aspect of web development that I enjoy most is identifying business processes with efficiency gaps and implementing lean, fully custom software solutions that fill those gaps

thinblueline.run Website

The Run/Walk for the Thin Blue Line is an annual charity run sponsored by OKCOPS. The 2018 run will be the 16th annual event. I volunteered my time to create this website and set up a giving form for the organization through Qgiv. I built this micro-site in the Summer of 2017 and continue to make updates.

The site features images from runs past, responsive design, a race results search feature, and details about the organization and next year's race. OKCOPS is a chapter of National COPS (Concerns of Police Survivors), a registered 501c3 non-profit organization. COPS provides support to family members of police officers who have been killed in the line of duty

Skills Used: JavaScript, CSS3, Responsive Design, Collaboration
